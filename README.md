# egg_project
一个使用egg.js实现的文章浏览h5页面  
包涵首页查看文章列表，点击某个文章详情，以及上传一个带图片的新的文章  
技术栈：   
后端 ---- nodejs(使用egg.js框架)  
前端 ----  vue 主框架；ui使用：vant  
ajax使用fetch  
跨域使用前端proxy代理

```
npm install
```
